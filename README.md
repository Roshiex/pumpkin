Pumpkin's Website

This is my first website as a project for web development at SETU. Visitors can read about Pumpkin, she how she rules the world, and show her love.



Features:
Home: Meet a cartoon of Pumpkin and see what she has to say about the website
About: Learn about Pumpkin, her background and her personality
Gallery: Polaroid images of Pumpkin in her journey
Food Review: Review of Pumpkin's favourite and least favourite foods along with paw ratings
History: This has been redacted
Geopolitics: Pumpkin's very correct takes on global events
Guestbook: Where visitors can leave their messages of love for Pumpkin


Layout uses CSS Flexbox and Grid
Custom typography using Google fonts (linked below)
Polaroid images and themed sections


Technologies Used:
HTML5
CSS3
Google Fonts 

Live Demo:
View website live at https://roshiex.github.io/pumpkin


Usage:
Open the website on Google Chrome. No installation needed
